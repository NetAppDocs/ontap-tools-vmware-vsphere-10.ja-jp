= ONTAP toolsでvCenter Serverのクレデンシャルを更新
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
メンテナンス コンソールを使用して、vCenter Server インスタンスの資格情報を更新できます。

.作業を開始する前に
メンテナンスユーザのログインクレデンシャルが必要です。

.タスクの内容
デプロイ後に vCenter Server の資格情報を変更した場合は、この手順を使用して資格情報を更新します。

.手順
. vCenter Serverで、VASA Providerへのコンソールを開きます。
. maintenance ユーザとしてログインします。
. と入力 `2` して[System Configuration Menu]を選択します。
. 入力 `8` vCenter の資格情報を変更します。
