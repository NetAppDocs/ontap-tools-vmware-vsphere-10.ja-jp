= ONTAP tools Managerのパスワードのリセット
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAPツール マネージャのパスワードを忘れた場合は、 ONTAP tools for VMware vSphereメンテナンス コンソールから生成されたリセット トークンを使用して管理者アクセスを復元できます。

.手順
. ウェブブラウザを開いて、 `https://<ONTAPtoolsIP>:8443/virtualization/ui/` ONTAPツール マネージャーにアクセスします。
. ログインページで、「パスワードのリセット」を選択します。
. ONTAP tools for VMware vSphereを使用して、パスワード リセット トークンを生成します。
+
.. vCenter Server にログインし、メンテナンス コンソールを開きます。
.. 入力 `2`*システム構成*を選択します。
.. 入力 `3`*「maint」ユーザーパスワードの変更*を選択します。


. パスワード リセット ダイアログで、リセット トークン、ユーザー名、新しいパスワードを入力します。
. 資格情報を更新するには、[*リセット*] を選択します。
. 新しいパスワードを使用してONTAPツール マネージャにログインします。
